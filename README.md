# Driven Analytics Dashboard

A beautiful, responsive analytics dashboard built for the Driven automotive streaming platform. This demo showcases engagement metrics across video content, social media, and user activity.

## üé® Features

- **KPI Cards**: Track key metrics like total views, active users, engagement rate, and sign-ups
- **Video Performance**: Multi-line chart showing video views and watch time trends
- **Social Media Analytics**: Bar chart displaying follower counts across platforms
- **Traffic Sources**: Doughnut chart breaking down where visitors come from
- **Weekly Activity**: Area chart showing daily session patterns
- **Top Content**: List of best-performing videos with view counts
- **Key Insights**: Actionable insights and recommendations
- **Fully Responsive**: Works beautifully on desktop, tablet, and mobile devices

## üõ†Ô∏è Setup Instructions

### Option 1: Quick Start (View Locally)

1. Download the `driven-dashboard` folder
2. Open `public/index.html` in your web browser
3. That's it! The dashboard will load with all mock data

### Option 2: Deploy to GitHub Pages (Free Hosting!)

1. **Create a new GitHub repository**
   - Go to https://github.com/new
   - Name it: `driven-dashboard`
   - Make it public
   - Don't initialize with README (we already have one!)

2. **Upload your files**
   - Click "uploading an existing file"
   - Drag and drop the contents of the `public` folder (index.html, dashboard.jsx, README.md)
   - Commit the files

3. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "Pages" in the left sidebar
   - Under "Source", select "main" branch
   - Click Save
   - Your dashboard will be live in 1-2 minutes!

4. **Access your dashboard**
   - URL will be: `https://[your-github-username].github.io/driven-dashboard/`
   - Share this link with anyone!

## üîß Customization

To update the data with real metrics:

1. Open `dashboard.jsx`
2. Find the `mockData` object at the top
3. Replace values with actual data from your analytics platforms
4. Save and refresh!

The dashboard automatically formats numbers, percentages, and calculates trends.
